- CSS Text Alignment
  - `text-align`
    - `left`
    - `center`
    - `right`
    - `justify` - to perfectly center all the text with left and right margins/edges.
  - `text-align-right` - this property specifies how to align the last line of a text/paragraph.
    - `left`
    - `center`
    - `right`
